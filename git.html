git 常用指令

  • git clone repo (下載專案)
  • git commit repo -m [message] (提交修改)
  • git push repo (丟到遠端專案)
  • git pull repo (更新本地專案)
  • git revert [commit hash] (返回某次提交結果)
  • git reset [commit hash] (將本地專案時間樹倒回某次提交之前)
  • git submodule add [repo] (加入子專案)
  • git checkout -b [to_branch] [from branch] (新增分支並移動到分支)
  • git checkout [branch] [移動到分支]
  • git restore file (移除尚未提交的檔案修改)
  • git status (本地當前專案狀態)

特殊場合

  1. 修改遠端歷史 (非必要不使用,會造成同事pull不了專案)
  • git rebase -i Head~N (修改 N個紀錄)
  • git push repo -f

git ssh 下載配置

  1. gen public key and private key from putty
  2. public key 放置 git 網站設定
  3. openssh 加入 private key
  4. git clone 測試
